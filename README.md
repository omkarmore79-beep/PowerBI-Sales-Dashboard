Superstore Sales Analytics Dashboard
Dataset Name: Superstore Sales Dataset

This dataset is perfect because it supports:
Time analysis
Profit vs sales
Region-wise comparison
Category performance

Project Overview
**The objective of this dashboard is to analyze sales performance, profitability, and customer segments across different regions and time periods to support business decision-making.

Key Features
KPI cards showing Sales, Profit, Orders, and Profit Margin
Category and Sub-category performance analysis
Geographic sales distribution
Customer segment contribution
Top product performance tracking
Interactive slicers and filters

Tools Used
Power BI
DAX
Power Query
Data Visualization Techniques

Business Insights
Identified top-performing product categories
Compared sales and profitability across sub-categories
Analyzed regional sales distribution
Evaluated customer segment contributions
